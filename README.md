AI-Driven Smart Waste Segregation & Sustainability Monitoring System
Intelligent Urban Waste Classification and Sustainability Intelligence Platform Using AI and IoT
📌 Project Overview

This project presents an AI-powered smart waste segregation system designed to classify waste materials into four categories:

Metal

Organic

Paper

Plastic

The system uses transfer learning with MobileNetV2 to automatically classify waste images and provide sustainability insights. It supports real-time monitoring, cloud integration potential, and environmental impact assessment.

🎯 Objectives

Develop an AI-based waste classification model

Achieve high validation accuracy with real-world robustness

Analyze misclassification patterns using confusion matrix

Provide sustainability insights and recycling efficiency metrics

Prepare system for IoT and web deployment

🧠 Model Architecture

Base Model: MobileNetV2 (Transfer Learning)

Input Size: 224x224

Output Classes: 4

Activation: Softmax

Loss Function: Categorical Crossentropy

Optimizer: Adam

📊 Model Performance

Training Accuracy: 98%

Validation Accuracy: 93%

Evaluation Tools:

Confusion Matrix

Precision / Recall / F1-Score
